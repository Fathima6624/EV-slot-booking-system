GoCharge is a Django-based web application for managing EV charging stations and booking charging slots.  
The system supports **role-based access** for users and station operators, with real-time slot availability and automatic expiry handling.

---

## Key Features

### User
- Browse EV charging stations
- View charging points and available slots
- Book and cancel slots
- View upcoming and completed bookings
- Download booking receipts

### Operator
- Operator dashboard with statistics
- Manage charging stations
- Add and manage charging points
- Create, activate, and deactivate slots
- Automatic slot expiry based on time
- View operator-specific bookings

---

## Tech Stack

- **Backend**: Django (Python)
- **Frontend**: HTML, CSS
- **Database**: SQLite
- **Authentication**: Django Auth
- **Timezone Handling**: Django timezone utilities
